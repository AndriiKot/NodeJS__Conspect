В Node.js вы можете удалить папку со всем содержимым асинхронно, 
используя модуль `fs` и методы, такие как `fs.promises.rmdir` 
или `fs.rm` (начиная с Node.js 12.10.0). Лучше всего использовать `fs.rm`, поскольку он поддерживает рекурсивное удаление. Вот пример кода, который показывает, как это сделать:

```javascript
const fs = require('fs/promises');

async function removeDirectoryRecursively(path) {
    try {
        await fs.rm(path, { recursive: true, force: true });
        console.log(`Директория ${path} успешно удалена`);
    } catch (error) {
        console.error(`Ошибка при удалении директории: ${error.message}`);
    }
}

// Используйте функцию
removeDirectoryRecursively('./папка_для_удаления');
```

### Объяснение кода:
1. Импортируется модуль `fs/promises` для работы с файловой системой с использованием промисов.
2. Функция `removeDirectoryRecursively` принимает путь к папке, которую нужно удалить.
3. Используется метод `fs.rm` с параметрами `{ recursive: true, force: true }` 
   для удаления папки и всего ее содержимого.
4. Если удаление успешно, выводится сообщение; в случае ошибки выводится сообщение об ошибке.

### Примечания:
- Убедитесь, что вы используете Node.js версии 12.10.0 или выше для использования `fs.rm`.
- Используйте `force: true`, если хотите игнорировать ошибки, 
  связанные с отсутствием файлов или папок.
