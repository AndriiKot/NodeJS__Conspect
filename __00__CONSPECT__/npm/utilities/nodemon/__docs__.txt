Nodemon — это утилита для Node.js, 
которая автоматически перезапускает ваш сервер или приложение, 
когда в файлах проекта вносятся изменения. 
Это позволяет разработчикам не беспокоиться 
о ручном перезапуске приложения, 
что значительно ускоряет процесс разработки.

Когда вы работаете над приложением на Node.js, 
обычно вам нужно вручную останавливать и запускать сервер, 
чтобы увидеть изменения. Nodemon наблюдает 
за изменениями в файловой системе 
и автоматически перезапускает сервер, ч
то делает процесс разработки более удобным и эффективным.

### Установка

Чтобы установить Nodemon, вы можете использовать npm (менеджер пакетов для Node.js):

```bash
npm install -g nodemon
```

Флаг `-g` устанавливает nodemon глобально, что позволяет запускать его из любого места на вашем компьютере.

### Использование

Для использования Nodemon вместо обычного Node.js, просто замените команду `node` на `nodemon` в терминале:

```bash
nodemon your-app.js
```

Где `your-app.js` — это ваш основной файл приложения.

### Примеры

Если вы хотите запустить приложение и 
следить за изменениями только 
в определённых файлах или директориях, 
вы можете использовать различные опции Nodemon. 
Например, чтобы следить только за файлами с расширением `.js`:

```bash
nodemon --ext js your-app.js
```

Nodemon также можно настроить через конфигурационный файл 
(например, `nodemon.json`), где можно указать различные параметры, 
такие как порты, директории и файлы для игнорирования.

Таким образом, Nodemon — это полезный инструмент, 
который облегчает разработку приложений на Node.js.
