node-readability - это npm-пакет, 
который предоставляет простой API для извлечения "читабельного" 
контента из HTML-документов. Он используется 
для очистки веб-страниц от ненужных элементов, 
таких как реклама, навигация, 
и выделения основного текстового содержимого.
