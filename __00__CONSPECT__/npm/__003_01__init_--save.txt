Строка npm i --save express является командой для установки 
пакета express с помощью менеджера пакетов Node Package Manager (npm).

Разберём по частям:

npm: это команда для работы с npm, 
который является менеджером пакетов для JavaScript,
 обычно используется с Node.js.
i: это сокращение для команды install, то есть "установить".
--save: этот флаг указывает npm 
добавить установленный пакет
 в список зависимостей в файле package.json. 
 Однако начиная с npm версии 5, эта опция по умолчанию включена, 
 поэтому использовать её не обязательно — достаточно просто написать npm install express.
express: это название пакета, который вы хотите установить. 
Express — это популярный веб-фреймворк для Node.js, 
который упрощает создание серверных приложений.
Таким образом, команда npm i --save express 
устанавливает пакет Express и добавляет его в зависимости вашего проекта.
